# v0.4.2 - LOG_MESSAGESの不足キーを補完

## 概要

内部で `logger.xxx()` として使用されているにもかかわらず `LOG_MESSAGES` に未登録だったキーを追加した。

## 変更内容

### 追加
- `src/messages/logger.ts` - 以下のキーを `LOG_MESSAGES` に追加
  - `COMMAND_NOT_FOUND`
  - `COMMAND_EXECUTION_FAILED`
  - `BUTTON_HANDLER_NOT_FOUND`
  - `BUTTON_EXECUTION_FAILED`
  - `LISTENER_EXECUTION_FAILED`
  - `VOICE_RECORDING_FAILED`

### 変更なし（破壊的変更なし）
