# v0.2.0 - リアーキテクチャ

## 概要

過剰抽象を全削除し、4つのBotが共通で使う基盤フレームワークとして再設計。

## 変更内容

### 削除
- `src/core/` 全体（ATACOrchestrator, ContextManager, EventRouter, PipelineExecutor, PluginLoader, ServiceContainer）
- `src/plugins/` 全体（BasePlugin, 各プラグイン実装）
- `src/events/` 全体（CronJobHandler, MessageMentionHandler, ReplyHandler, SlashCommandHandler, WebhookHandler）
- `src/config/` 全体（ConfigLoader）
- `src/services/` 全体（各Adapter, 各Interface）
- `src/types/` 全体（旧型定義）
- `src/utils/` 全体（旧ユーティリティ）
- `src/main.ts`, `src/bootstrap.ts`
- `config/customers/` 全体

### 追加
- `src/bot/client.ts` - createBot() + BotClient
- `src/bot/command.ts` - Command + CommandRegistry + ButtonRouter
- `src/bot/listener.ts` - Listener型 + ListenerRegistry
- `src/interaction/reply.ts` - ReplyBuilder
- `src/interaction/button.ts` - ButtonRowBuilder
- `src/interaction/embed.ts` - EmbedHelper
- `src/interaction/poll.ts` - PollBuilder
- `src/voice/recorder.ts` - VoiceRecorder
- `src/voice/types.ts` - 録音関連型
- `src/env/loader.ts` - loadEnvironment()
- `src/messages/errors.ts` - ATACError + createError
- `src/messages/logger.ts` - Logger
- `src/types/bot.ts` - Bot設定・ライフサイクル型
- `src/types/command.ts` - コマンド関連型
- `src/types/listener.ts` - リスナー関連型

### 依存関係変更
- `discord.js`, `zod` を peerDependencies に移動
- `dotenv` を dependencies に追加
- 不要パッケージ削除: firebase-admin, @pinecone-database/pinecone, ai, @ai-sdk/*, @upstash/redis, node-cron
- `"type": "module"` 追加（ESM化）
