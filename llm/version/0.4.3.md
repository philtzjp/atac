# v0.4.3 - 外部エラーコード登録機能の追加

## 概要

ATACを使用するプロジェクトが独自のエラーコードを登録できるよう、`registerErrorCodes()` 関数を公開した。
`registerLogMessages()` と同じパターン。

## 変更内容

### 追加
- `src/messages/errors.ts` - `registerErrorCodes(codes: Record<string, string>)` 関数
- `src/index.ts` - `registerErrorCodes` のエクスポート

### 変更なし（破壊的変更なし）
